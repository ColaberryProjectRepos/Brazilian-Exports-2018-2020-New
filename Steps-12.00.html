Add Title and Logos 12


1. Add a textbox by navigating to the Insert section of the Home tab and click on Text box

2. Type Brazilian Exports 2018-2020 in Segoe UI font with size 24 font

3. Center the title and make it bold

4. Expand the format pane on the right, click on the Properties tab, and click on the Size tab. Expand the Position tab and set both Horizontal and Vertical to 0 Set Height to 73 and Width to 1279
 

5. Click on the Insert tab, navigate to the Elements section, and click on Image. Insert the Colaberry logo

6. Expand the Format pane, click on the General tab

7. Expand the Size tab and set Height to 52 and Width to 205. Expand the Position tab and set Vertical to 19


8. Click on the Insert tab, navigate to the Elements section, and click on Image. Insert the Brazil flag

9. Expand the Format pane, click on the General tab

10. Expand the Size tab and set Height to 73 and Width to 122. Expand the Position tab and set Horizontal to 1157
