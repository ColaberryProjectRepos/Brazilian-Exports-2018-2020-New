Add Filled map 18


1. Click on Filled map and add Region and Country to Location

  • Add Total Export Earnings, Export Earnings Per Month, Export Concentration, Export Earnings MoM, and Export Earnings YoY to Tooltips, and rename them Total EE, EE Per Mth, EC, EE MoM%, and EE YoY respectively




2. Click on the Format visual tab, click on the General tab, and expand the Properties tab
  • Expand the Position tab, set Horizontal 730 and Vertical to 397
  • For Size, set Height to 322 and Width to 549
  • Expand the Title tab, change Text to Export Destination, Change Text color to #fbf12e, Background color to #44AA29, and Horizontal alignment to Center.

3. Expand the Visual tab next to General, expand the Fill colors tab and click the fx conditional formatting icon
  • Set Format style to Gradient
  • Set What field should we base this on? to Total Export Earnings
  • Set Lowest value color to #fdf9ab and Highest value color to #7e7917


4. Click on the Insert tab next to the Home tab and click on Image, expand the General tab
  • In the Size tab set Height to 61 and Width to 138
  • Expand the Position tab, set Horizontal to 743 and Vertical to 606

5. Click on Text box
  • In the Size tab set Height to 34 and Width to 46
  • Expand Position and set Horizontal to 756 and Vertical to 648
  • Expand Background, set color to#A6C8E5
  • Click inside the Text box and set font to Segoe UI, Font size 10, Center alignment. Type Low inside the Text box

6. Click on Text box
  • In the Size tab set Height to 36 and Width to 52
  • Expand Position and set Horizontal to 829 and Vertical to 648
  • Expand Background, set color to#A6C8E5
  • Click inside the Text box and set font to Segoe UI, Font size 10, Center alignment. Type High inside the Text box
