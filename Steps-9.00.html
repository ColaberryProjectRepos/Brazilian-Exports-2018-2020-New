Load data into Power BI 9


1. Open Power BI Desktop and click on Get Data

2. Search for SQL Server Database in the Database tab or the All tab and open it by either double clicking it or clicking the green Connect button.


3. Copy and paste your Server name from SQL Server (the text inside the Server name prompt from the small window that opens every time you open SQL Server) into the Server prompt and click on the OK button

4. A Navigator window will open. Here you will expand the Brazilian Exports tab and select all 5 tables


5. Now click on Transform Data which will open Power Query Editor

6. Navigate to the Queries tab on the left and click on Dim_Date. Click on the Add Column tab at the top of the screen, hover over to Column From Examples and click on the text, a dropdown will appear, click on From All Columns

7. Fill in the first row of Column 1 with 1/1/2018, this will fill all the remaining rows in Column 1. Change 1/1/2018 in the fourth row to 2/1/2018, this will change the remaining rows to the values we need. Rename the column that appears from Merged to Date

8. Right click on the Date column header and Change Type to Date

9. Click on the arrow in the Date column header and click on Sort Ascending

10. Click on File and click on Close & Apply
