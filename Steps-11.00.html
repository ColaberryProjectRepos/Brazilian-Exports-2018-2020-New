Create Measures Table and Measures 11


1. Click on Enter Data in the Home tab

2. Rename Column 1 to Measures and name the table Measures Table


3. Right click on the Measures Table and click on New measure

4. Enter the formula for Total Export Earnings, 
 
Total Export Earnings = SUM(Fact_BrazilExports[ExportEarnings])
 
, into the DAX formula bar.

5. Right click on the Measures column, click on Delete from model and click on Yes


6. Create the following measures by typing the following DAX formulas into the DAX formula bar:
 
Export Earnings Per Month = DIVIDE([Total Export Earnings], COUNTROWS(SUMMARIZE('Calendar', 'Calendar'[Month], 'Calendar'[Year])))
-----------------------------------------------------------------------------------------
Export Concentration = 
DIVIDE( CALCULATE([Total Export Earnings], ALL(), VALUES(Dim_Product), 
            VALUES(Dim_BrazilLocation[State]), VALUES(Dim_ExportDestination[Region]), 
            VALUES(Dim_ExportDestination[Country]), VALUES('Calendar'[Date])
        ),
        CALCULATE([Total Export Earnings], ALL(), VALUES('Calendar'[Year])
        )
)
 
Click on Measure tools at the top. Format this measure as Percentage and set it to 1 decimal place.
-----------------------------------------------------------------------------------------
Export Earnings MOM =
 
VAR ExportEarningsLM = CALCULATE([Total Export Earnings], DATEADD('Calendar'[Date], -1, MONTH))
 
RETURN
 
IF(ISBLANK(ExportEarningsLM), 0, DIVIDE([Total Export Earnings]- ExportEarningsLM, ExportEarningsLM))
 
Click on Measure tools at the top. Format this measure as Percentage and set it to 2 decimal places.
-----------------------------------------------------------------------------------------
Export Earnings YOY = 
 
VAR ExportEarningsLY = CALCULATE([Total Export Earnings], DATEADD(Calendar[Date], -1, YEAR))
 
VAR ExportEarningsCY = CALCULATE([Total Export Earnings], DATEADD('Calendar'[Date], 0, YEAR), YEAR('Calendar'[Date]) <> 2018 )
 
RETURN
 
IF(ISBLANK(ExportEarningsLY), 0, DIVIDE(ExportEarningsCY - ExportEarningsLY, ExportEarningsLY))
 
Click on Measure tools at the top. Format this measure as Percentage and set it to 2 decimal places.

Arrow EA MoM = SWITCH(TRUE(), [Export Earnings MOM] <0, UNICHAR(9660), [Export Earnings MOM] >0, UNICHAR(9650))
----------------------------------------------------------------------------------------------
Arrow EA YoY = SWITCH(TRUE(), [Export Earnings YOY] <0, UNICHAR(9660), [Export Earnings YOY] >0, UNICHAR(9650))
