Add Arrow Indicators 14


1. Select the Card visualization and drag Arrow EA MoM measure and drop it onto the Card

2. In the Visualizations pane click on the Format your visual tab. Expand the Callout value tab and set Font size to 16. Below Color click on the fx Conditional Formatting button. Set Format style to Rules. Set What color should we base this on? to Export Earnings MoM  If value > 0 and <= Max then color #44AA29, If value >= Min and < 0 then color #CD4C46, if Value = 0 then color White

3. Set Category value to Off

4. Click on the General tab, next to Visual, and expand the Properties tab. Expand the Size tab and set Height to 30 and Width to 35. Expand the Position tab and set Horizontal to 168 and Vertical to 518


5. Repeat the process above for the Arrow EA YoY measure

6. Font size 16 and Conditional formatting for color. Set What color should we base this on? To Export Earnings YoY If value > 0 and <= Max then color #44AA29, If value >= Min and < 0 then color #CD4C46, if Value = 0 then color White
 
7. Set Category label to Off

8. Click on the General tab next to Visual, and expand the Properties tab. Expand the Size tab and set Height to 30 and Width to 35. Expand the Position tab and set Horizontal to 168 and Vertical to 634
