Add Shape map 19


1. Click on Shape map and add State from Dim_BrazilLocation to Location
  • Add Total Export Earnings to Color saturation and rename it to Total EE
  • Add Export Earnings Per Month, Export Concentration, Export Earnings MoM, and Export Earnings YoY to Tooltips and rename them as Total EE, EE Per Mth, EC, and EE MoM respectively
 
2. Click on the Format visual tab, click on the General tab, expand the Properties tab, and expand the Position tab
  • Set Horizontal 206 and Vertical to 397
  • For Size, set Height to 322 and Width to 523
  • Expand the Title tab, change Text to State, Change Text color to #fbf12e, Background color to #44AA29, and Horizontal alignment to Center.
 
3. Expand the Visual tab next to General and expand Map settings
  • Change Map type to Brazil: states
  • Expand the Zoom subtab and set Manual zoom to On.
     
4. Expand Fill colors, set the Minimum color to #F8D8C1 and set the Maximum color to #C75B0C


5. Click on the Insert tab next to the Home tab and click on Image, expand the General tab
  • In the Size tab set Height to 93 and Width to 138
  • Expand the Position tab, set Horizontal to 581 and Vertical to 589

6. Click on Text box
  • In the Size tab set Height to 33 and Width to 49
  • Expand Position and set Horizontal to 591 and Vertical to 650
  • Click inside the Text box and set font to Segoe UI, Font size 10, Center alignment. Type Low inside the Text box.

7. Click on Text box
  • In the Size tab set Height to 33 and Width to 47
  • Expand Position and set Horizontal to 662 and Vertical to 650
  • Click inside the Text box and set font to Segoe UI, Font size 10, Center alignment. Type High inside the Text box
