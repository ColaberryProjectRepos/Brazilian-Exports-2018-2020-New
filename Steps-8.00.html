Import Fact and Dimension tables to SQL Server 8

1. Download the Brazilian Exports Project Essentials by clicking on the red button above

2. Open SQL Server and navigate to the Object Explorer

3. Hover over the Databases folder and right-click on it

4. Click on the Import Data-tier Application option

5. The Import Data-tier Application wizard will open, click Next. In the Import Settings tab you’ll be asked to “Specify the BACPAC to import.” Click on Browse and navigate to the folder containing the .bacpac file and click on the BrazilianExports_2018_2020.bacpac file. Click on Next

6. In the Database Settings tab, you’ll be asked to provide a new database name. Name it ”Brazilian Exports” and click on Next.

7. In the Summary tab click on Finish

8. After the import finishes click on Close.



9. The Brazilian Exports database should be visible when you expand the databases tab in SQL Server.
