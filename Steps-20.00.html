Postscript Addition 20


Oftentimes you’ll find that changes and/or additions need to be made to a report to improve it. These may be brought to your attention by the report end users or by yourself. Nevertheless, updating a report should be an expected part of report development. This step is an example of that. In this step we’ll be adding a measure to the Measures table and then we’ll use that measure to create a sort of sub-KPI that’ll allow us to State Export Concentration, we’ll essentially use the State field in the denominator of our Percentage calculation so that the Grand Total is calculated at a state level and not at the national level.

You create the measure by typing the following DAX formula into the DAX formula bar:

State Export Concentration =

IF(HASONEVALUE(Dim_BrazilLocation[State]),

    DIVIDE(CALCULATE([Total Export Earnings], ALL(), VALUES(Dim_Product),

            VALUES(Dim_BrazilLocation[State]), VALUES(Dim_ExportDestination), VALUES('Calendar'[Date])

            ),

            CALCULATE([Total Export Earnings], ALL(), VALUES('Calendar'[Year]),

                VALUES(Dim_BrazilLocation[State])

            )

    )

)

 

Format State Export Concentration as a percentage with one decimal place.

Then you expand the Visualizations pane, and in the Build visual tab select the Multi-row card. Drag and drop the State Export Concentration measure into the Multi-row card. Navigate to the Fields prompt and rename State Export Concentration to State Export Conc.

Click on the Format visual tab, expand Callout values, and set the font to DIN and size 14. Set Category labels to Off. Click on the conditional formatting, fx, button next to Color. Select Rules for Format style, and select State Export Concentration for What field should we base this on? Define the rule, If value is blank then white.

 

Click on the General tab and expand Properties, set Height to 61 and Width to 201, expand Position and set Horizontal to 203 and Vertical to 658. Set Title to On. Expand Title and type in State Export Conc. In the Text prompt. Select Font DIN and size 14. Click on the conditional formatting, fx, button next to Text color. Select Rules for Format style, and select State Export Concentration for What field should we base this on? Define the rule, If value is blank then white.
