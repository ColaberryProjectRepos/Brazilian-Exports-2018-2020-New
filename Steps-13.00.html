Add Date Slicer and KPI Cards 13


1. In the Build visual tab of the Visualizations pane click on Slicer

2. Drag the Year field from Calendar and drop it on the slicer

3. Click on the Format your visual tab and expand the Slicer header tab. Change the font size to 14. Expand the Slider tab and change the color to Black. Expand the Values tab and Bold the text

4. Click on the General tab next to Visual, and expand the Properties tab. Expand the Size tab and set Height to 104 and Width to 204. Expand the Position tab and set Horizontal to 0 and Vertical to 71

5. Switch Title to On and expand the Title tab. Under Text type Year, change Text color to #fbf12e, change Background color to #44aa29, and change Horizontal alignment to Center


6. In the Build visual tab of the Visualizations pane click on Card

7. Drag the Total Export Earnings measure from the Measures table and drop it on the Card

8. Click on the Format your visual tab and expand the Callout value tab. Change font size to 38 and change Color to #BCB523. Set Category label to Off

9. Click on the General tab next to Visual, and expand the Properties tab. Expand the Size tab and set Height to 112 and Width to 204. Expand the Position tab and set Horizontal to 0 and Vertical to 175

10. Switch Title to On and expand the Title tab. Under Text type Total Export Earnings, change Text color to #fbf12e, change Background color to #44aa29, and change Horizontal alignment to Center


11. Follow the same process for the remaining Cards. 
 
12. Export Earnings Per Month: Visual -> Callout Value -> Font -> Size - 38, Color - #112E93, Category Label - Off. General -> Properties -> Size -> Height: 111 Width: 204, General -> Properties -> Position -> Horizontal: 0 Vertical: 287
Switch Title to On, Title -> Title -> Text - Export Earnings Per Month, Font - 13, Text color - #fbf12e, Background color - #44aa29, Horizontal alignment - Center.
 
13. Export Concentration: Visual -> Callout Value -> Font -> Size - 38, Color - #B498F4, Category Label - Off. General -> Properties -> Size -> Height: 92 Width: 204, General -> Properties -> Position -> Horizontal: 0 Vertical: 398
Switch Title to On, Title -> Title -> Text - Export Earnings Per Month, Font - 14, Text color - #fbf12e, Background color - #44aa29, Horizontal alignment - Center.
 
14. Export Earnings MoM%: Visual -> Callout Value -> Font -> Size - 38, Color - #AAE1C9, Category Label - Off. General -> Properties -> Size -> Height: 114 Width: 204, General -> Properties -> Position -> Horizontal: 0 Vertical: 490
Switch Title to On, Title -> Title -> Text - Export Earnings MoM% Font - 14, Text color - #fbf12e, Background color - #44aa29, Horizontal alignment - Center.
 
15. Export Earnings YoY%: Visual -> Callout Value -> Font -> Size - 38, Color - #f1b183, Category Label - Off. General -> Properties -> Size -> Height: 114 Width: 204, General -> Properties -> Position -> Horizontal: 0 Vertical: 605
